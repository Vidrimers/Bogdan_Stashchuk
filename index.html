2. Переменные

Переменные дают возможность повторного доступа к значениям

myName = "Yaro" - контейнер для хранения значений

Имена переменных

PascalCase - каждое имя с заглавной буквы. Для типов и классов
DB_PASSWORD - Все большие буквы. значения известны до запуска приложения и не меняются
camelCase - первая буква в нижнем регистре

Объявление переменных

let a; - Объявление переменной
const c = 10; - Объявление и присваивание
a = true; - Присваивание
let a = 10;
a = 20; - переобъявление, в итоге будет выводиться 20
let b; - пока значение не присвоено, будет выводиться undefined
b = false;
CONST
const c = 10; - нужно присваивать сразу
c = 20; - если константу присваивать позже, то будет выводиться ошибка //TypError: Assignment to constant variable
Объявление и присваивание
console.log(a); - будет выводиться ошибка ReferenceError: a is not defined, потому что переменная не объявлена
let a; - объявим
console.log(a); - будет выводиться undefined, потому что значение не присвоено
a = true; - присвоим
console.log(a); - будет выводиться true

3. Примитивные типы

Тип переменной определяется типом присвоенного значения

string (строка) // 'строка'
boolean (логический) // true false
number (число) // 1 2 3 1.5 0.5 -1 1.25
null // - когда значение пока отсутствует,
undefined // значения у переменной нет
symbol (символ) // используется редко
const a = 10; - число // number
const b = 'abc'; - строка // string
